
Юутуб (ПО Українськи) — від Furry
--------------------------------
Файли в архіві:
- index.html — головна сторінка
- styles.css — стилі
- server.js — маленький express-сервер щоб зберігати користувачів (users.json)
- users.json — база користувачів (початково пусто)
- package.json — щоб запустити сервер: npm install && npm start
- app/myapp.apk — місце для вашого apk (файлу нема — додайте свій)

Застереження:
- Вбудований ключ YouTube API знаходиться у index.html. Це небезпечно для публічного розміщення — краще зберігати ключ на сервері.
- Ця демо-версія підходить для локального тестування. Щоб мати реальний production-сайт, потрібно налаштувати бекенд, безпечне зберігання ключів, та ліцензійні питання.
